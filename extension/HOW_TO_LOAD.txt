SPEEDY AI EXTENSION - HOW TO LOAD
==================================

CHROME:
-------
1. Open Chrome and go to: chrome://extensions/
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the "dist" folder
5. Done! The extension is now loaded

FIREFOX:
--------
1. Open Firefox and go to: about:debugging#/runtime/this-firefox
2. Click "Load Temporary Add-on..."
3. Navigate to "dist-firefox" folder
4. Select the "manifest.json" file
5. Done! The extension is now loaded

NOTES:
------
- Chrome uses: dist/ folder
- Firefox uses: dist-firefox/ folder
- Both are built separately to avoid manifest conflicts
- To rebuild: npm run build:chrome or npm run build:firefox
